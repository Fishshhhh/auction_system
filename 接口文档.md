# 竞拍系统API接口文档

## 基础配置模块

### 资产分类管理

#### 1. 获取所有资产分类
- **URL**: `/api/config/categories`
- **Method**: `GET`
- **Description**: 获取所有启用状态的资产分类
- **Response**:
```json
[
  {
    "id": 1,
    "name": "二手设备",
    "code": "EQUIPMENT",
    "description": "各类二手设备",
    "status": 1,
    "createdTime": "2023-01-01T12:00:00",
    "updatedTime": "2023-01-01T12:00:00"
  }
]
```

#### 2. 创建资产分类
- **URL**: `/api/config/categories`
- **Method**: `POST`
- **Description**: 创建新的资产分类
- **Request Body**:
```json
{
  "name": "新分类名称",
  "code": "NEW_CODE",
  "description": "分类描述",
  "status": 1
}
```
- **Response**: 返回创建成功的分类对象

#### 3. 更新资产分类
- **URL**: `/api/config/categories/{id}`
- **Method**: `PUT`
- **Description**: 更新指定ID的资产分类
- **Request Body**:
```json
{
  "name": "更新后的名称",
  "code": "UPDATED_CODE",
  "description": "更新后的描述",
  "status": 1
}
```
- **Response**: 返回更新后的分类对象

#### 4. 删除资产分类
- **URL**: `/api/config/categories/{id}`
- **Method**: `DELETE`
- **Description**: 删除指定ID的资产分类
- **Response**: 204 No Content

#### 5. 搜索资产分类
- **URL**: `/api/config/categories/search`
- **Method**: `GET`
- **Query Parameters**: `keyword` (搜索关键词)
- **Description**: 根据关键词搜索资产分类
- **Response**: 返回匹配的分类列表

### 竞拍模板管理

#### 1. 获取所有竞拍模板
- **URL**: `/api/config/templates`
- **Method**: `GET`
- **Description**: 获取所有启用状态的竞拍模板
- **Response**:
```json
[
  {
    "id": 1,
    "name": "公开竞价模板",
    "auctionType": 1,
    "configParams": "{\"startPrice\": 10000, \"bidIncrement\": 500}",
    "description": "公开竞价模板描述",
    "isDefault": 1,
    "status": 1,
    "createdTime": "2023-01-01T12:00:00",
    "updatedTime": "2023-01-01T12:00:00"
  }
]
```

#### 2. 创建竞拍模板
- **URL**: `/api/config/templates`
- **Method**: `POST`
- **Description**: 创建新的竞拍模板
- **Request Body**:
```json
{
  "name": "新模板",
  "auctionType": 1,
  "configParams": "{\"startPrice\": 10000, \"bidIncrement\": 500}",
  "description": "模板描述",
  "isDefault": 0,
  "status": 1
}
```
- **Response**: 返回创建成功的模板对象

#### 3. 更新竞拍模板
- **URL**: `/api/config/templates/{id}`
- **Method**: `PUT`
- **Description**: 更新指定ID的竞拍模板
- **Request Body**:
```json
{
  "name": "更新后的模板",
  "auctionType": 1,
  "configParams": "{\"startPrice\": 15000, \"bidIncrement\": 1000}",
  "description": "更新后的描述",
  "isDefault": 0,
  "status": 1
}
```
- **Response**: 返回更新后的模板对象

#### 4. 删除竞拍模板
- **URL**: `/api/config/templates/{id}`
- **Method**: `DELETE`
- **Description**: 删除指定ID的竞拍模板
- **Response**: 204 No Content

#### 5. 根据类型获取竞拍模板
- **URL**: `/api/config/templates/type/{type}`
- **Method**: `GET`
- **Description**: 获取指定类型的竞拍模板
- **Response**: 返回匹配类型的模板列表

## 竞拍类型说明

| 类型值 | 类型名称 |
|-------|---------|
| 1 | 公开实时竞价 |
| 2 | 暗拍 |
| 3 | 无底价拍卖 |
| 4 | 定向拍卖 |
| 5 | 降价拍卖 |
